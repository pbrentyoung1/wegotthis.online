# Prompting Guide

## Purpose

Store reusable guidance for prompting AI assistants on this project.

## Current Decisions

- Prompts should include project context, current stack direction, and instruction not to invent final decisions.
- Prompts should point assistants to `/docs`.

## Open Questions

- Which prompts should become reusable templates?
- What context is required for design, architecture, implementation, and review prompts?

## Notes

- Keep prompts concise enough to be reused and specific enough to prevent drift.
