### 3. Branded Checkboxes and Radios on Forms Demo

The forms demo uses Inspinia's `form-checkbox` and `form-radio` classes (not `form-check-input`). These classes apply `checked:bg-primary!` and `checked:border-primary!`, giving checkboxes and radios the ForWorship brand color instead of browser-default blue.

### 4. Flatpickr Date Pickers on Forms Demo

Date inputs in the forms demo use:

```html

```

Flatpickr is already included in `app.js` and auto-initializes on `data-provider="flatpickr"` inputs. Calendar icon spans have `pointer-events-none` so clicks pass through to the input.

### 9. Typography Page Responsive Fix

File: `resources/views/ui/typography.blade.php`

The grid uses `xl:grid-cols-6` (6 columns at 1280px+). All `col-span` values were using `lg:` prefix, which activated at 1024px — before the grid had multiple columns. This caused implicit column creation and broken layout between 1024–1279px.

Fixed: all `lg:col-span-X` changed to `xl:col-span-X` throughout the page.
